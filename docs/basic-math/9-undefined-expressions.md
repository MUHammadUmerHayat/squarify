## Word Problem Question:

$
\text{ }
$

$
 \text{A) } \Large \frac{5}{0}

$

$
\text{C) } \Large \frac{0}{5}

$
  {\LARGE \frac{5}{0}} \large \text{ because the denominator is } 0
